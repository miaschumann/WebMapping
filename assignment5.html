Welcome to Washington DC!

This is a map to help you select activities for your visit.

On the interactive map, you can find music, bars, theatre, and, of course, bikes to help you get to each of these locations. Filter by activity type using the blue menu.

Click on the icons to learn more. Although you might not find much useful information in the popups because this is actually an assignment designed to learn how to make versitile layouts for different screen sizes as well as bounds, ensuring the user does not navigate away from Washington DC.
